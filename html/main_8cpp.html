◆ anotherTask()

void anotherTask ( void *  p_params)

A fairly useless demonstration task function.

This function is run by the RTOS scheduler at a higher priority than loop(), so it can interrupt loop() when it needs to. It just prints an asterisk once every two seconds.

Parameters
p_paramsA pointer to any parameters passed to this task; there aren't any, so this value is ignored

◆ loop()

void loop ( )

Arduino's low-priority loop function.

A non-RTOS Arduino program runs all of its continuously running code in this function after setup() has finished. When using FreeRTOS, loop() implements a low priority task on most microcontrollers, and crashes on some others. This demonstration function reads the optical encoder and prints its position.

void setup ( )

Arduino setup function which runs once at program startup.

This function sets up a serial port for communication and creates the tasks which will be run. Hardware such as an optical encoder is also set up in this function.
